Current app capabilities

SmartManager currently supports a LAN-first server and client setup, plus owner recovery, messaging workflows, restaurant tools, and reporting.

  • Store Server Setup

    Run the store server on the main store machine with LAN access and VPN support for managers.

  • Electron Desktop App

    Use the desktop client on Windows, macOS, or Linux and connect it to your store server over LAN or VPN.

  • Remote Manager Access

    Owners and managers can connect over VPN, while cashiers remain limited to the store network.

  • POS, Inventory, and Receipts

    Run sales, products, suppliers, customers, stock control, low-stock views, receipts, and daily store operations.

  • Messaging and Campaign Outbox

    Use the current build’s bulk messaging templates, campaigns, and outbound communication workflows.

  • Owner Recovery and Offline Codes

    Recover the owner account on the server computer and regenerate offline recovery codes when needed.

  • Licensing, Seats, and Devices

    Base licenses, seat entitlements, device-bound login, and server-side device admission are already enforced.

  • Restaurant, Reports, and Setup

    Manage kitchen tickets, tables, and orders, run POS and restaurant reports, and configure receipts, taxes, staff, and backups.

Start with the right connection setup

Keep cashier devices on the store LAN, use VPN only for approved manager access, and run owner recovery on the server computer when needed.
